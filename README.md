# 🚀 Web3 Early Builder

A simple and powerful Web3 project to track testnet tasks, airdrops, and early builder activities.

## 🔥 Features
- Track testnet participation
- Manage airdrop tasks
- Stay consistent in Web3
- Beginner-friendly system

## 📂 Project Structure
- tasks.json → store tasks
- app.js → main logic
- README.md → project info

## 🛠️ Tech Stack
- JavaScript
- Node.js

## 🚀 Getting Started
1. Clone the repo
2. Run the project
3. Start tracking tasks

## 💡 Goal
To help Web3 users stay active and increase chances of airdrop rewards.
